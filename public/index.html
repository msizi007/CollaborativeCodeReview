CodeReview Platform API

Welcome to the backend landing page. Use the endpoints below in Thunder Client or Postman.

Authentication (/api/auth)

POST/registerCreate a new account
POST/loginObtain JWT Token

Projects (/api/projects) PROTECTED

GET/Get all projects
POST/Create a project
GET/:idGet project details
PUT/:idUpdate project info
DELETE/:idRemove project
GET/:id/membersList project members
POST/:id/membersAdd user to project
DELETE/:id/members/:uidRemove user from project

code Submissions (/api/submissions)

GET/List all code submissions
POST/Upload new code for review
GET/:idView specific submission
PATCH/:id/statusUpdate status (e.g., Approved/Pending)
GET/:id/commentsGet feedback on code
POST/:id/commentsAdd review comment

Users (/api/users)

GET/Get all users
GET/:idUser profile info
PUT/:idUpdate user data
DELETE/:idDelete account

Global Comments (/api/comments)

PUT/:idEdit a comment
DELETE/:idRemove a comment
